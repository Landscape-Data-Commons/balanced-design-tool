This tool is a uses the implementation of the Generalized Random Tesselation Stratified (GRTS) algorithm from spsurvey to draw spatially balanced sample designs.

To report bugs, contact Nelson Stauffer at nelson.stauffer@ars.usda.gov or open an issue in the GitHub repository.

The packages used are:

  • ggplot2
  • leaflet
  • dplyr
  • rgdal
  • sf
  • shiny
  • spdplyr
  • spsurvey
  • viridis
