This tool is a Shiny interface for an R session that uses an implementation of the Generalized Random Tesselation Stratified (GRTS) algorithm to draw spatially balanced sample designs.

To report bugs, contact Nelson Stauffer at nelson.stauffer@ars.usda.gov

The packages used are:

  • rgdal

  • shiny

  • sp

  • spdplyr

  • spsurvey

  • tidyverse
